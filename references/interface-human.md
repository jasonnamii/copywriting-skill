# Interface: Human-Skill Consumption
## How copywriting-engine activates 16 axes + 44 meta-principles

---

## COGNITIVE AXES (7) → READING EXPERIENCE DESIGN

### Axis C1: Attention & Focus (Executive Function)
**Copy Application:** Controls information hierarchy. How much at once? How scannable?

**S1 vs S2 Processing:**
- **S1 (Fast, intuitive):** High-impact visual hierarchy. Big number first. Emotional color. "10x faster" → eyes stop.
- **S2 (Slow, deliberate):** Dense evidence. Multi-step logic. "Here's why..." → requires concentration.

**Copy Technique:**
- S1 design: Lead with number/result, then details.
  - KR: "📈 주간 매출 3배 상승" (눈에 띄는) vs 설명문 먼저
  - EN: "$50k revenue. One month. Here's how." vs "We implemented a system that..."
- S2 design: Start with problem, build to evidence.
  - KR: "왜 기존 방법이 안 됐는지 먼저 설명하고 나서 해법 제시"
  - EN: Problem context → research data → mechanism → result

**Implementation:** For A1-A3 (onboarding): S1 dominant (visual, quick wins). For B1-B2 (strategy): S2 dominant (reasoning, evidence chains).

---

### Axis C2: Working Memory (Cognitive Load)
**Copy Application:** Sentence length, list structure, information density.

**Reading Experience:**
- Sentence length ≤ 15 words = faster S1 processing
- Lists > paragraphs (working memory can hold 3-4 items at once)
- One concept per sentence

**Copy Technique:**
- KR Bad: "우리의 최첨단 인공지능 기술은 당신의 복잡한 편집 작업을 자동화함으로써 창의력을 증대시키고 생산성을 극대화합니다"
- KR Good: "AI가 편집합니다. 당신은 창의력에만 집중합니다. 결과? 3배 더 만듭니다."
- EN Bad: "Our proprietary machine learning system automates your editing workflow by reducing cognitive friction through intelligent optimization"
- EN Good: "AI edits. You create. 3x output."

**Implementation:** Read aloud. Pause = too long. Split sentence.

---

### Axis C3: Language Processing (Semantic Access)
**Copy Application:** Word choice, jargon vs plain language, metaphor effectiveness.

**Principle:** High-frequency words load faster than low-frequency.

**Copy Technique:**
- Avoid jargon unless audience speaks it natively
- Use concrete nouns (creator, video, audience) vs abstract (stakeholder, content)
- Metaphors work if they're audience-native (creator = builder, community = network)

Examples:
- KR Jargon: "프로프라이어터리 알고리즘의 임베딩 레이어" (creator 모르는 말)
- KR Plain: "우리만의 AI 방식으로" (충분)
- EN Jargon: "leveraging algorithmic optimization matrices"
- EN Plain: "Using AI to..."

**Implementation:** When writing copy, flag any word non-creator-native would pause on. Replace with 1-2 grade simpler.

---

### Axis C4: Decision-Making (Choice Architecture)
**Copy Application:** Number of options, default selection, how choice is framed.

**Framing Effects:**
- **Gain frame:** "Choose the plan that adds X value"
- **Loss frame:** "Avoid missing out on X value"
- **Neutral frame:** "Compare the options"

**Copy Technique:**
- CTA with gain frame (growth focus): "Unlock 10x faster exports"
- CTA with loss frame (pain avoidance): "Stop wasting 2 hours daily on edits"
- Multiple options: Show 3 max (paradox of choice kills conversion)

Examples:
- KR Gain: "당신의 잠재력을 50% 더 꺼내보세요"
- KR Loss: "매일 낭비되는 2시간을 되찾으세요"
- EN Gain: "Unlock 5x more creator time"
- EN Loss: "Stop losing 120 hours yearly to editing"

**Implementation:** Test gain-frame vs loss-frame variants. Loss typically wins for pain-focused products; gain for aspiration-focused.

---

### Axis C5: Memory & Retrieval (Encoding)
**Copy Application:** How memorable is the copy? Does it stick?

**Techniques for memorability:**
- Vivid, specific detail (brain encodes concrete > abstract)
- Emotional connection (remembered better)
- Pattern interruption (unexpected stands out)
- Repetition of core claim (3x in different phrasings)

**Copy Technique:**
- Specific detail: "Malia went from 5-hour edits to 12-minute edits"
- General detail: "Save time editing" (forgotten in 10 sec)
- Repetition pattern:
  1. Headline: "30-minute edits"
  2. Subhead: "12 minutes average with AI"
  3. CTA: "See your time saved now"

Examples:
- KR Vivid: "지난주 금요일 오후 3시, 처음 AI로 편집했을 때의 그 짜릿한 기분 있잖아?"
- KR Generic: "편집이 빨라집니다"
- EN Vivid: "Yasmin was drowning in 200 unedited clips. Now she publishes 10/week."
- EN Generic: "Faster editing"

**Implementation:** Every major claim needs a concrete example/detail. Vivid beats generic 3:1 on recall.

---

### Axis C6: Predictive Processing (Mental Models)
**Copy Application:** Does copy align with audience's existing schema (belief system)?

**Principle:** Copy that confirms existing beliefs is more persuasive than copy that contradicts.

**Copy Technique:**
- **Align with schema:** "Like successful creators, you're thinking about growth"
- **Shift schema gently:** "You're probably optimizing for views. Smart. But [expert/data] says audience loyalty matters more."
- **Use analogies from their world:** "It's like your editing timeline—drag and drop"

Examples:
- KR Aligned: "프리랜서라면 매출 변동성이 고민이시죠? (creator 현실인정)"
- KR Schema shift: "매출은 안정적인데 팀을 왜 못 키우나? (새로운 관점)"
- EN Aligned: "As a creator, you know consistency wins."
- EN Schema shift: "Posting more doesn't build loyalty. Community does."

**Implementation:** Profile audience's existing mental model first. Write to confirm, then gently expand.

---

### Axis C7: Abstraction & Transfer (Concept Learning)
**Copy Application:** Can audience apply learning elsewhere? Does copy teach or just sell?

**Principle:** Copy that teaches transfers better than copy that only pitches.

**Copy Technique:**
- Teach principle first, then product application
- KR: "좋은 콘텐츠 3가지: 정보, 감정, 재미. 우리 AI는 [재미] 부분을 당신이 다시 확인하게 해줍니다"
- EN: "Great editing has 3 principles: pace, emotion, pacing. Our tool handles pacing so you decide emotion."

**Implementation:** For B1-B2 (strategy), C6 (email education), C8-C9 (thought leadership): Always teach principle before pitching solution.

## MOTIVATION AXES (2) → CTA DESIGN

### Axis M1: Autonomy (Self-Determination Theory)
**Copy Application:** CTA language that respects independence, not controls.

**Anti-patterns (Demotivating):**
- "You must," "You should," "You have to," "Required"
- "Complete this task," "Sign up now" (commanding)

**Pro-patterns (Autonomy-respecting):**
- "Discover," "Explore," "Try," "See for yourself"
- "Join others who," "Become," "Unlock"

**Copy Technique:**
- KR Controlling: "지금 가입해야 합니다"
- KR Autonomous: "당신의 성장을 시작해보세요"
- EN Controlling: "You must sign up now"
- EN Autonomous: "See what's possible for you"

**Implementation:** Every CTA should feel like an invitation, not a command. Replace imperatives with invitations.

---

### Axis M2: Goal Pursuit (Commitment & Consistency)
**Copy Application:** Urgency and commitment language. Small commitments lead to large.

**Techniques:**
- Start with smallest commitment (download, 30-sec quiz, free trial)
- Make initial commitment public (share result)
- Frame escalation as natural progression

**Copy Technique:**
- Start: "Try free for 14 days. No credit card."
- Escalate: "Most users then upgrade. See why." (pattern, not pressure)
- Commit: "Join [count] creators who've made this part of their weekly flow"

Examples:
- KR Escalation: "체험하고 나면? 80%는 첫 달에 무료를 유료로 전환합니다. (자연스러운 진행)"
- EN Escalation: "One free export usually leads to premium. Here's why customers stay."

**Implementation:** Design copy in 3-step commitment ladder, not single giant ask.

## SOCIAL AXES (3) → SOCIAL PROOF & BELONGING COPY

### Axis S1: Conformity (Normative Influence)
**Copy Application:** "Join X others," "The standard now," "Everyone's doing..."

**Effectiveness depends on:**
- Is claim true? (False conformity destroys trust)
- Is it relevant to audience? (Conformity to creators matters to creators; conformity to accountants doesn't)
- Is it group they want to join? (Aspirational group > demographic)

**Copy Technique:**
- KR: "상위 1% 크리에이터들이 사용 중입니다" (aspirational group)
- KR (weaker): "한국인 70% 가 사용합니다" (demographic, less persuasive)
- EN: "Used by creators at every level" (inclusive)
- EN: "Used by 2 million people" (generic, weak)

**Implementation:** "Successful creators" > "2M users." Aspirational group > large number.

---

### Axis S2: Identity (Self-categorization Theory)
**Copy Application:** "For creators like you," "Your type," appeal to self-image.

**How it works:**
- Product positions are identity signals. Choosing = self-definition
- Identity copy DOES NOT NAME PRODUCT. Names audience instead.

**Copy Technique:**
- KR Identity play: "프리랜서지만 기업처럼 (정체성 선언)"
- KR Product play: "우리 도구를 씁니다" (약함, identity 아님)
- EN Identity: "For creators building empires, not just posting"
- EN Product: "For people using our software" (weak, not identity)

**Examples:**
- KR: "단순히 좋아요를 받으려는 게 아니라, 비즈니스를 하는 크리에이터라면 (정체성 기반)"
- EN: "If you see yourself as a creator-preneur, not just content maker..." (identity-focused)

**Implementation:** Start with "For [identity]..." not "For users of..." Identity > demographics.

---

### Axis S3: Status & Prestige (Costly Signaling)
**Copy Application:** Premium/exclusive positioning, price as quality signal, high-status association.

**When effective:**
- Audience aspires to higher status
- Product is (or can be) genuinely premium
- Design/presentation supports prestige

**Copy Technique:**
- Exclusivity: "Invitation-only," "Limited to [number]," "By application"
- Quality signal: Price + prestige (higher price = higher quality perception)
- Association: "Trusted by [high-status brand/person]"

Examples:
- KR Prestige: "베타 멤버 40명만. 선착순." (배타성)
- KR Prestige + design: "심플함은 비싼 디자인의 특징입니다. 마찬가지로..."
- EN Prestige: "Used by founders at Y Combinator companies"
- EN Prestige: "The expensive choice. Worth every penny."

**Implementation:** Status copy MUST align with visual design. Design must feel premium. Otherwise copy feels inauthentic.

## EVOLUTION AXES (2) → PRIMAL TRIGGERS

### Axis E1: Vigilance (Threat Detection)
**Copy Application:** Urgency, scarcity, threat-framing. Without manipulation.

**Ethical activation:**
- Scarcity is REAL (limited spots, time-bound)
- Threat is REAL (competitors, opportunity cost)
- Frame is honest (not manufactured panic)

**Copy Technique:**
- Real scarcity: "Beta closes Dec 31. 18 spots left of 100."
- Real threat: "Your competitors are already [doing X]. See where that's heading."
- Opportunity cost: "Every month you wait is X revenue missed"

Examples:
- KR Ethical: "12월이 마지막 50% 할인이에요. 내년엔 정가입니다. (사실)" vs "거의 다 팔렸어요" (거짓)
- EN Ethical: "Annual price increases Jan 1. This rate expires Dec 31." vs "Everyone's buying!" (hype)

**Implementation:** Scarcity copy: Always provide number/deadline. Make it verifiable.

---

### Axis E2: Mating Signals (Status Signaling, Refined)
**Copy Application:** High-status association, quality indication, aspirational positioning.

**Difference from S3 (Status):**
- E2 is primal (unconscious quality judgment)
- E3 is social (conscious group belonging)
- E2 works through association, not explicit claim

**Copy Technique:**
- Association: "As used by..."
- Refinement signal: "Simple. Elegant. Built for people who care about craft."
- Quality markers: Natural language, attention to detail in copy itself (no typos, no corporate-speak)

Examples:
- KR Quality signal: "복잡함을 버렸습니다. (다듬음의 신호) vs "우리의 고급 기능을" (과장)"
- EN Quality signal: "Obsessed with simplicity." (craft marker)
- EN Bad: "Our cutting-edge AI-powered platform leverages..." (hype, not quality)

**Implementation:** Show refinement through copy quality itself. Short, elegant sentences. No hype language.

## EMOTION AXIS (1) → EMOTIONAL TONE

### Axis Em1: Emotional Contagion (Emotional Transmission)
**Copy Application:** Transmit feeling through language, testimonials, imagery.

**Techniques:**
- **Emotional specificity** beats abstract feeling
- **Authentic voice** beats polished corporate
- **Sensory detail** activates mirror neurons
- **Real person quote** (name, photo) beats corporate claim

**Copy Technique:**
- KR Contagious emotion: "처음 영상이 1만 뷰를 넘었을 때의 그 떨리는 기분... (감정전염)"
- KR Generic: "더 많은 사람이 봅니다" (감정 없음)
- EN Contagious: "I was shaking when the numbers hit 100k. Real."
- EN Generic: "Increased reach" (no feeling)

**Emotion × Type Matrix:**
| Copy Type | Emotion | Example |
|-----------|---------|---------|
| A1 (onboarding) | Excitement + Confidence | "You're about to feel like a pro" |
| C1 (brand) | Pride + Belonging | "Join creators who take craft seriously" |
| C3 (landing) | Hope + Possibility | "What you've been imagining is finally possible" |
| C6 (email) | Recognition + Care | "We see you grinding. Here's what helps." |
| D1 (retention) | Gratitude + Momentum | "Thank you for the trust. Here's more." |
| D2 (upsell) | Ambition + Envy (positive) | "Watch them grow faster than you thought possible" |

**Implementation:** Every major copy section should trigger one core emotion. Name it. Write to that feeling, not just idea.

## BEHAVIOR AXIS (1) → HABIT FORMATION COPY

### Axis B1: Behavioral Inertia (Friction Reduction)
**Copy Application:** Default effects, habit stacking, how to make new behavior feel old.

**Techniques:**
- **Reduce friction:** Fewer steps, shorter decision time
- **Habit stacking:** "Like you already do with [existing habit]..."
- **Make default easy:** "Most people skip this. Don't. It takes 2 min and saves 2 hours."

**Copy Technique:**
- KR Friction reduction: "로그인 필요 없습니다. URL 붙여넣으세요. 10초."
- KR Habit stacking: "인스타그램에서 스토리 올리듯이" (기존 습관과 연결)
- EN Friction: "No sign-up. No credit card. Click once. Done."
- EN Habit stack: "If you know how to like a post, you can do this"

**Default effect copy:**
- Don't ask "Want to enable notifications?" → Default ON
- Copy should frame default as smart choice: "Notified weekly (you can adjust anytime)" not "Opt in for notifications"

Examples:
- KR Default: "매주 월요일 팁을 받으세요 (기본값). 관심 없으면 끄면 돼요."
- EN Default: "Weekly tips arrive Monday. Adjust anytime." (frames as feature, not obligation)

**Implementation:** For A2-A3 (onboarding), B4 (internal), D1-D3 (lifecycle): Reduce visible steps. Default to helpful behavior.

## META-PRINCIPLE CONSUMPTION

When 2+ axes are active, these meta-principles guide copy decisions:

### Top 10 Meta-Principles for Copywriting

1. **Specificity Beats Generality**
   - "3,489 creators, 23% avg growth" > "Many creators see more followers"
   - Activates: C5 (Memory), C6 (Pattern recognition)
   - Example: KR "주간 평균 1.3시간 절감" > "시간 절감"

2. **Emotion Before Logic**
   - Lead with feeling, support with reason
   - Activates: Em1 (Contagion), S2 (Identity)
   - Example: "Felt alive creating again. The tool helped because..." > "The tool improves..." + emotion

3. **Concreteness Over Abstraction**
   - "Sarah's videos got 10k views in 2 days" > "More visibility"
   - Activates: C3 (Language processing), C5 (Memory)
   - Example: KR "영상을 올린 지 2일 만에 5만 뷰" > "많은 사람이 봐요"

4. **Show, Don't Tell**
   - User result, not product claim
   - Activates: S1 (Conformity through example), Em1 (Contagion)
   - Copy should include result-focused testimonials, not feature lists

5. **Reverse the Expectation**
   - Counterintuitive hooks drive 34% more engagement
   - Activates: C4 (Decision), Em1 (Surprise)
   - Example: "You're probably doing it wrong" > "Here's how to do it right"

6. **Use Social Currency**
   - Make sharing rewarding to the sharer (ego, belonging, practical help)
   - Activates: S2 (Identity), S1 (Conformity)
   - Example: "Share your growth secret" (ego) > "Share our tool" (product)

7. **Activate Loss First, Then Gain**
   - Scarcity + pain avoidance > aspiration alone
   - Activates: Hit-skill Axis 2 (Loss aversion)
   - Structure: "Avoid losing [X]" then "Unlock [Y]"

8. **Default to Simplicity**
   - When copy is hard to parse, it's not believed
   - Activates: C2 (Working memory), C3 (Language)
   - Implementation: Read aloud. If you pause, simplify.

9. **Make the Audience the Hero**
   - Not the product, not the founder
   - Activates: S2 (Identity), M1 (Autonomy)
   - Example: "You're the creator. This is your tool." not "We're the best platform."

10. **Proof Beats Persuasion**
    - Data + testimonial > best rhetoric
    - Activates: C5 (Memory), S1 (Conformity)
    - Always include: 1 metric + 1 quote in major copy block

## INTEGRATION PROTOCOL

### When to invoke human-skill analysis before writing copy

**High-stakes copy (ALWAYS):**
- C3: Landing pages (make or break conversion)
- C4: Ads (expensive, need axis precision)
- D1: Retention (highest lifetime value impact)

**Medium-stakes (FOR NEW CAMPAIGNS):**
- C5: Social/community
- C6: Email sequences
- B1: Strategy docs (internal persuasion)

**Low-stakes (OPTIONAL, use if brief unclear):**
- A9: Loading states
- B4: Internal reports
- C8: Thought leadership (creator-native, often intuitive)

### Decision tree for copy approach

```
Brief received
  → Identify copy type (A/B/C/D)
  → High-stakes? YES → Run full 16-axis analysis
               NO → Quick check: What's the primary emotion/motivation?
  → Match axes to copy strategy
  → Draft copy (L1 focus)
  → Check meta-principles (at least top 3 relevant ones)
  → Iterate with C8 mode (if creator-focused) or standard mode
  → Final output: Copy brief + 3 variants
```

### Example: C4 Ad Copy Brief Output

```
TYPE: C4 (Ads)
Audience: Creators seeking growth (identity = ambitious builder)

PRIMARY AXES:
- C1 (Attention): Hook with big number first
- Em1 (Emotion): Excitement about possibility
- E1 (Vigilance): Competitors moving faster
- M1 (Autonomy): This is YOUR growth, not ours

SECONDARY AXES:
- S1 (Conformity): Proof of adoption
- C5 (Memory): Vivid detail (specific creator result)

META-PRINCIPLES:
1. Specificity (name + number)
2. Emotion before logic
3. Loss + gain frame
4. Proof beats persuasion

COPY STRUCTURE:
Headline: Hook with emotion + number (S1 processing)
  "Watch creators hit 1M followers (without selling their souls)"
Subhead: Remove objection (identity preservation)
  "Growth that feels like you, not like a machine"
Body: Story + proof (Em1 + Specificity)
  "Maya went from posting to her 10k niche to 250k in 6 months. Her secret? Consistency, not clickbait. We just removed the editing friction."
CTA: Autonomy-framed (M1)
  "See what's possible for you. Try free."

NOTES:
- Lead with emotion (excitement), support with data
- No corporate language
- CTA should feel like invitation, not command
- Test loss frame variant: "Stop losing creators to tools that feel inauthentic"
```

## QA CHECKLIST: Before shipping copy

- [ ] Primary axis is activated (emotional or motivational impact felt)
- [ ] One specific metric included (not just "faster" or "better")
- [ ] One authentic quote/testimonial (real person, context)
- [ ] Sentence length < 15 words average
- [ ] No jargon outside audience's native language
- [ ] CTA is framed as invitation, not command
- [ ] Emotion conveyed before or with logic
- [ ] Shareability hook included (if social/email context)
- [ ] For C8: Conversational tone, creator-specific pain acknowledged, actionable insight
- [ ] For high-stakes (C3, C4, D1): 2+ axes intentionally activated
