# Cross-Tone System: 4-Dimension Tone Framework

## Overview
Tone is a 4-dimensional space. Every copy type occupies a position on each dimension. Position varies by mode (C8 vs Universal).

## 4 Tone Dimensions

### 1. Funny ↔ Serious
- **Funny end**: Jokes, wordplay, absurdist humor, pop culture references
- **Serious end**: Clinical, authoritative, no-nonsense, factual
- **Neutral (50)**: Conversational but grounded

### 2. Casual ↔ Formal
- **Casual end**: Contractions, slang, short sentences, "hey/yo/lol"
- **Formal end**: Complete sentences, professional vocabulary, grammatically strict
- **Neutral (50)**: Standard conversational English/Korean

### 3. Irreverent ↔ Respectful
- **Irreverent end**: Challenges status quo, questions authority, breaks rules deliberately
- **Respectful end**: Honors conventions, defers to expertise, acknowledges hierarchy
- **Neutral (50)**: Polite but candid

### 4. Enthusiastic ↔ Matter-of-Fact
- **Enthusiastic end**: Exclamation marks, superlatives, energetic adjectives, CAPS
- **Matter-of-fact end**: Understated, measured, data-driven, no hype
- **Neutral (50)**: Normal energy level

## Tone Conflict Resolution

### When Type Default Clashes with Situational Need

**Scenario 1: Error Message (Serious type) Needs Humor**
- C8 mode allows +20 Funny if it reduces user frustration
- Rule: Humor only if it's about the situation, never about the user's mistake
- Example: "Oops, our servers went for a coffee break. Back in 5." ✓
- Example: "You messed up again, lol" ✗

**Scenario 2: Feature Launch (Enthusiastic) Needs Matter-of-Fact Tone**
- Reduce Enthusiastic by -15 if it's a complex/serious feature
- Rule: Use Enthusiastic for benefit, Matter-of-Fact for mechanics
- Example: "Ship 3x faster. Here's how →" (first sentence enthusiastic, second methodical)

**Scenario 3: Social Post (Casual) Needs Formal Context**
- Situation: Responding to government/PR query on social
- Reduce Casual by -20, increase Respectful by +20 for that reply only
- Core post stays casual; replies flex formal

### Tone Consistency Rules

1. **Within-page gradient** (most important)
   - Hero: Full type default tone
   - Body: -5 to -15 points Casual/Funny (introduce complexity)
   - CTA: +5 points Enthusiastic (urgency)

2. **Across funnel stages**
   - Awareness: Higher Funny/Casual (get attention)
   - Consideration: -10 Funny, -5 Casual (build credibility)
   - Conversion: +10 Enthusiastic (create urgency)

3. **Cross-channel consistency**
   - Same message across email, push, social should maintain ±10 on each dimension
   - Email can be -5 Casual vs social (literacy difference)
   - Push can be +15 Casual vs email (brevity)

## Tone Modulation Rules: Page Gradient

### Hero Section (100% tone)
Example C8 C6 Social Post (Funny 55, Casual 80, Irreverent 60, Enthusiastic 80):
- "We built the TikTok for your to-do list. Yes, really."
- Maintain all 4 dimensions at max

### Body Section (-10 to -15 complexity)
- Reduce Funny by 10 (humor becomes rare, focused)
- Reduce Casual by 5 (add complete sentences)
- Maintain Irreverent/Enthusiastic (intrigue + benefit still matter)
- Example: "Most to-do apps are boring asf. This one? It's actually fun. Watch creators ship faster. [video]"

### CTA Section (+10 Enthusiastic)
- Maintain or increase Enthusiastic
- Reduce Funny (laser focus)
- Keep Casual/Irreverent from body
- Example: "Sign up free → Let's go!"

## Mode Trigger Rules

**C8 Mode Activates If:**
- Copy brief includes "C8", "Cre8orClub", or "creator tone"
- Product is creator/influencer/community-first
- Tone should default to hip/informal until stated otherwise

**Universal Mode Activates If:**
- No C8 keyword present
- B-type (always Universal)
- Standard professional context

**Mode Lock:** Once mode is set for a campaign, all 27 types use that mode (don't mix).

## Metrics by Dimension

- **Funny:** Click-through on humorous variant vs. straight variant (CTR lift)
- **Casual:** Engagement (shares, replies) on casual vs. formal
- **Irreverent:** Retention of "rule-breaker" vs. "establishment" message
- **Enthusiastic:** Conversion rate on energetic vs. measured message

Target: Test 1 dimension per type per campaign. Rotate through 4 over time.
