## 11법칙 = 카피 통과 체크리스트

| # | 법칙 | 카피 적용 |
|---|------|----------|
| 1 | 관련 낱말을 함께 배열 | 주어-동사 사이에 절 삽입 ✗ |
| 2 | 강조어 = 문장 끝 | "쌍둥이봉과 같다" — 핵심 단어 마지막 |
| 3 | 추상 → 구체 | "신선한" → "파닥파닥" / "맑은" → "15초·진실" |
| 4 | 동사 정곡 + 부사 절제 | "매우/너무나/대단히" 금지. "어슬렁거렸다 ≫ 천천히 걷는다" |
| 5 | '것' 중복 금지 | 한 카피에 '것' ≤ 1회 |
| 6 | '도'·'등' 자제 | 열거 시 동의어·생략으로 변화 |
| 7 | 주어 반복 금지 | "그/그녀/당신" 한 단락 1회 |
| 8 | 단어·어미 중복 금지 | "생각했다 → 의견을 나누었다" |
| 9 | 과잉 수사 금지 | "굵직굵직 → 굵직" / "사회적인 → 사회적" |
| 10 | 한 문장 = 한 주어·한 의미 | 이중 주어 ✗ |
| 11 | 자신없는 표현 줄이기 | "~인 것 같다 / ~라고 한다 / 언젠가" 0개 |

## 추가 4축 (축약·단문)

| # | 룰 | 카피 적용 |
|---|---|---|
| 12 | 한자투 정리 | "탑승하고 있던" → "탄" |
| 13 | 과잉 감정 금지 | "안타까운 시선이 쏟아졌을 정도" → 사실로 대체 |
| 14 | 빼도 좋을 조사 빼라 | "사람의 마음" → "사람 마음" / "10시 이후에" → "10시 이후" |
| 15 | 문장 허리 끊어라 | 22자+ 문장은 마침표로 분할 |
| 16 | 단문 연쇄 | 접속사 "이에 따라/특히/한편" 절제 |
| 17 | 시조 압축 | "텅 빈 산에 나뭇잎은 떨어지고" → "빈 산 잎 지고" |

## 카피 송출 직전 17체크 (KIWI Gate)

```
[ ] 1. 추상어 ≤ 2개 (구체어 우선)
[ ] 2. 동사 = 정곡, 부사·형용사 절제
[ ] 3. 강조어 문장 끝 배치
[ ] 4. '것/도/등/그' 빈도 = 각 ≤ 1회
[ ] 5. 같은 단어·어미 중복 0
[ ] 6. 한 문장 = 한 주어
[ ] 7. 자신없는 표현 0
[ ] 8. 감정 어휘 절제 (사실 대체)
[ ] 9. 조사 빼고 의미 보존 → 빼라
[ ] 10. 평균 ≤ 22자
[ ] 11. 한자투 정리
```

미달 시 카피 재작성. 산문 기준 통과 → 50%, 카피 기준(≤15자 헤로) 통과 → 100%.

## 카피 vs 산문 차이

KIWI 11법칙은 **산문 기준**. 카피는 더 강도 높은 압축:

| 산문 | 카피 |
|------|------|
| 22자 문장 | 7~15자 헤드 |
| 동사 1개 | 동사 0~1개 |
| 단락 5문장 | 헤드 1줄 + 서브 1줄 |
| 추상어 ≤ 2 | 추상어 ≤ 1 |

## §2.7 PRE_WRITE 5룰과 결합

KIWI 11법칙 = 한국어 자연성 / PRE_WRITE 5룰 = 카피 직조 = 둘 다 통과 = 형 자산.

**충돌 시 우선순위:**
1. PRE_WRITE 단문 (≤15자) > KIWI 11 (≤22자) — 카피는 더 짧게
2. PRE_WRITE 단일약속 > KIWI 16 단문연쇄 — 약속 1개가 우선
3. KIWI 14 조사 빼기 > PRE_WRITE 시그니처 — 조사 절제로 시그니처 강화

## WRONG/CORRECT 1쌍

❌ WRONG: "여러분의 피부는 매우 소중한 것입니다. 그래서 우리는 진정한 케어를 위해 여러 가지 노력을 다해 왔다고 할 수 있습니다."
- KIWI 위반: 추상어(여러분/소중한/진정한) · '것' · '도' · 자신없는 표현(할 수 있습니다) · 22자+

✅ CORRECT: "피부를 깊이 섬긴다. — 설화수"
- KIWI 통과: 동사(섬긴다) · 8자 · 단일약속 · 추상어 0
