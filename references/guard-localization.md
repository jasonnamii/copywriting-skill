# Guard: Localization (Korean ↔ English Re-Creation)

**Core Principle:** Copy localization is re-creation, not translation. A direct translation of compelling English copy becomes stilted Korean, and vice versa. Both languages require independent optimization from the same brief.

---

## Why Translation Fails for Copy

### The 4 Failure Points

1. **Cultural Context**
   - English: "Don't leave money on the table" resonates with entrepreneur mentality
   - Korean equivalent: "밥그릇을 내던지다" (throwing your rice bowl away) has darker, more desperate tone
   - Same concept, incompatible emotional payload

2. **Humor & Rhythm**
   - English wordplay (puns, alliteration) rarely survives translation
   - Korean relies on syllable count (4-syllable phrases: 단순명확한), particle placement for musicality
   - Direct translation destroys the rhythm structure

3. **Word Length & Reading Speed**
   - English average word: 4.7 letters, reads left-to-right linearly
   - Korean average word: 2-3 syllables, but Sino-Korean vs. native Korean affects perceived sophistication
   - Same message takes 30% more Korean space, changing scannability

4. **Tonal Mapping Complexity**
   - English: tone set mainly by word choice + punctuation (rare: exclamation marks = lower trust)
   - Korean: tone set by grammar endings (존댓말/반말), particles, Sino-Korean density, sentence structure
   - A single tonal layer in English maps to 3-4 in Korean

## Korean Copy Rules

### 존댓말 Spectrum: When to Use Each

| Form | Translation | Use Case | Tone Signal |
|------|-------------|----------|------------|
| 합니다체 | "하겠습니다" | Financial, legal, government, premium B2B | Formal, distant, trustworthy, old-fashioned |
| 하십시오체 | "하십시오" | Instruction/command, technical | Authoritative, demanding, formal |
| 해요체 | "해요" | B2C consumer, friendly B2B, app onboarding | Warm, approachable, modern, trustworthy |
| 해체 | "해" | Social, Gen-Z, casual brand, C8 mode | Friend-like, hip, accessible, insider language |
| 반말 | "해" (between close people) | Community, fan-facing, crisis/raw authenticity | Intimate, vulnerable, can feel immature if wrong |

**Rule:** Tone template specifies form. Violating (e.g., 해체 in financial copy) = immediate credibility loss.

### Particle Manipulation for Rhythm

**은/는 vs 이/가**
- 은/는: Topic marker, broader, conversational, softer
  - "디자인은 중요해요" (Design, it's important)
- 이/가: Subject marker, specific, emphatic, harder
  - "디자인이 중요해요" (Design IS important)

**Particle dropping for speed**
- Full: "이 도구가 당신의 작업을 더 빠르게 만들어줘요"
- Dropped: "이 도구, 당신 작업을 더 빠르게. 정말 빨라요" (snappier, C8 energy)

**Common rhythm particles:**
- 도 (too): adds layer of possibility
- 는데 (but/and): soft transition, creates flow
- 만 (only): emphasis through limitation
- 나 (or): creates choice energy

### Sino-Korean vs Native Korean Word Choice

**Sino-Korean (한자어):**
- Longer, more formal, sounds educated, feels distant
- "기능성" (functionality), "최적화" (optimization), "혁신적" (innovative)
- Risk: Sounds like translation (anti-pattern)

**Native Korean (순우리말):**
- Shorter, more emotional, sounds authentic, feels close
- "기능" (feature), "맞춤" (customized), "새로운" (new)
- Risk: Can sound unsophisticated in premium contexts

**Decision Rule:**
- B types (strategy): 80% Sino-Korean, 20% native (formal = trust)
- A types (UI): 60% native, 40% Sino-Korean (clarity + warmth)
- C/D types: 50/50, weighted to context (e.g., C8 = 70% native)
- Never >80% Sino-Korean in C8 mode (reads as translated corporate)

### Anti-Patterns: Korean Copy That Sounds Translated

| Red Flag | Bad Example | Why It Fails | Fix |
|----------|------------|-------------|-----|
| Overuse of 하다 | "서비스를 제공하다" | Sino-Korean 과다, 서면체 | "서비스를 드려요" |
| Stiff 존댓말 | "사용자께서 설정하실 수 있습니다" | 과도한 경어, 로봇 같음 | "설정할 수 있어요" |
| No particles | "나는 도구를 사용했다" | 문어체, 감정 없음 | "나 이 도구 쓰는데 정말 좋아" |
| Literal English structure | "더 빠르게 일하세요" | 한국식 절차 무시 | "더 빠르게 일할 수 있어요" |
| Mixed formality | "이 기능은 정말 awesome해요" | Tone whiplash | "이 기능, 진짜 미쳤어요" (C8) or "뛰어나요" (formal) |

## English Copy Rules

### Active Voice Dominance

- **Do:** "You save 3 hours per week"
- **Avoid:** "3 hours per week can be saved"
- English readers expect subject-verb-object clarity. Passive voice = credibility loss, perceived weakness

### Sentence Length Norms

- Average: 12-16 words per sentence (for web copy)
- Range: 5 words (punchy) ↔ 20+ words (rare, for emphasis)
- Never exceed 25 words without clear reason
- Pattern: Mix short + long for rhythm

**Anti-pattern:** All long sentences = academic, hard to scan. All short = listy, low credibility.

### Contractions & Formality Signals

- Use contractions (can't, don't, you're) = 80% of consumer copy
- Avoid contractions = 100% B-type, premium services, legal/medical
- No contractions in same piece = formal signal (can't mix)

### Cultural References That Don't Cross Borders

| Reference | Issue | Alternative |
|-----------|-------|-------------|
| "Like a Super Bowl ad" | Meaningless outside US/Canada | "Like a viral moment" |
| "It's not rocket science" | Assumes space program is reference | "It's simple" |
| British humor (dry, sarcasm) | Misread as cold/mean by US audiences | Test before shipping |
| Sports metaphors | American football ≠ global | Use universal sports (soccer) or remove |

## Parallel Creation Workflow

**Never translate. Create independently from the same brief.**

### Phase 1: Unified Brief (Both Languages)

Document only these elements:
- Core message (1 sentence, concept-level)
- Emotional target (what feeling should user have?)
- CTA intent (what action do we want?)
- Audience persona (tone spectrum)
- Constraint (length, platform, urgency)

**Example Brief:**
```
Core: "Design tools should help you think faster, not slower"
Emotion: Relief + capability ("I can finally do this")
CTA: Try free for 14 days
Persona: Designer, mid-career, frustrated with complexity
Length: Headline + subheader (total 10-15 words English, 8-12 words Korean)
Platform: Landing page hero
```

### Phase 2: Independent English Creation

1. Brainstorm 5+ headlines from brief
2. Select the strongest
3. Write native English structure (not translated-from-Korean structure)
4. Test: Read aloud. Does it flow?
5. Violate English rules intentionally? (If yes: justify or fix)

### Phase 3: Independent Korean Creation

1. Brainstorm 5+ Korean headlines from brief (not from English)
2. Select strongest (may sound nothing like English headline)
3. Choose 존댓말 form based on tone template
4. Add particles for rhythm
5. Decide Sino-Korean/native split
6. Test: Read aloud. Does it flow in Korean cadence?

### Phase 4: Comparison & Selection

- Line them up side by side
- Do they convey the same brief? (Answer: probably not identical, and that's OK)
- Is each the strongest version in its language?
- Does each feel native to the language?

**Decision:** Choose based on:
- If A/B test: may pick English headline but different Korean headline
- If single deployment: pick the strongest in each language (they may be different "styles")

## 10 Example Pairs: Translation vs Re-Creation

### Pair 1: Figma-Style Design Tool

**Bad Translation Approach:**
- English: "Ship designs faster"
- Korean (translated): "디자인을 더 빠르게 배송하세요" ❌ (sounds like shipping physical goods)

**Re-Created Approach:**
- English: "Ship designs faster"
- Korean: "디자인, 빨리 완성해요" ✓ (더 빠르게 마무리)

---

### Pair 2: Developer Tool

**Bad Translation:**
- English: "No config required"
- Korean (translated): "설정 필요 없습니다" ❌ (stiff, formal, confusing)

**Re-Created:**
- English: "No config required" (or: "No setup. Just build.")
- Korean: "설정? 필요 없어요" ✓ (conversational, 해요체)

---

### Pair 3: Premium Productivity App

**Bad Translation:**
- English: "Your second brain"
- Korean (translated): "당신의 두 번째 뇌" ❌ (awkward metaphor, not Korean idiom)

**Re-Created:**
- English: "Your second brain"
- Korean: "생각을 놓치지 않는 공간" ✓ (Don't lose your thoughts)

---

### Pair 4: E-commerce Guarantee

**Bad Translation:**
- English: "We've got your back"
- Korean (translated): "우리는 당신의 등을 지킵니다" ❌ (nonsensical)

**Re-Created:**
- English: "We've got your back"
- Korean: "뭔가 안 맞으면 환불해요" ✓ (Direct value: if wrong, refund)

---

### Pair 5: Onboarding (C8 Mode)

**Bad Translation:**
- English: "Vibe check: Is this tool for you?"
- Korean (translated): "분위기 확인: 이 도구는 당신을 위한 것인가?" ❌ (kills the vibe)

**Re-Created:**
- English: "Vibe check: Is this tool for you?"
- Korean: "이 도구, 너한테 어울릴 것 같아?" ✓ (conversational, 해체)

---

### Pair 6: Error Message Recovery

**Bad Translation:**
- English: "Oops! Something went wrong"
- Korean (translated): "앗! 뭔가 잘못되었어요" ❌ (too cutesy, undermines trust)

**Re-Created:**
- English: "Oops! Something went wrong"
- Korean: "일시적 오류가 발생했어요. 다시 시도해주세요" ✓ (technical, reassuring)

---

### Pair 7: Social Proof

**Bad Translation:**
- English: "Loved by 100K creators"
- Korean (translated): "100K 크리에이터에게 사랑받습니다" ❌ (stiff 존댓말)

**Re-Created:**
- English: "Loved by 100K creators"
- Korean: "크리에이터 10만 명이 쓰고 있어요" ✓ (active, present-tense energy)

---

### Pair 8: Feature Benefit (Sino-Korean Heavy)

**Bad Translation:**
- English: "Optimize your workflow"
- Korean (translated): "당신의 작업 흐름을 최적화하세요" ❌ (too formal for consumer copy)

**Re-Created:**
- English: "Optimize your workflow"
- Korean: "업무 흐름을 단순화했어요" ✓ (Sino-Korean OK here, but softer delivery)

---

### Pair 9: CTA (Time-Sensitive)

**Bad Translation:**
- English: "Grab your free trial now"
- Korean (translated): "지금 당신의 무료 평가판을 가져가세요" ❌ (no urgency)

**Re-Created:**
- English: "Grab your free trial now"
- Korean: "지금 시작해보세요 (14일 무료)" ✓ (actionable, playful, concrete timeframe)

---

### Pair 10: Mission/Vision Statement (B Type)

**Bad Translation:**
- English: "We believe design should be accessible"
- Korean (translated): "우리는 디자인이 접근성이 있어야 한다고 믿습니다" ❌ (잘못된 문법, 과도한 Sino-Korean)

**Re-Created:**
- English: "We believe design should be accessible"
- Korean: "모두가 디자인할 수 있는 세상을 만들어요" ✓ (합니다체, native Korean, active vision)

## C8 Mode Localization: Hip Korean ≠ Hip English

### English Hip Signals
- Lowercase (even proper nouns): "figma" not "Figma"
- Contractions + slang: "lemme," "gonna"
- Irreverent tone: Breaks rules intentionally
- References: Memes, internet culture, gaming
- Structure: Short sentences, fragments

### Korean Hip Signals (C8)
- 해체 + 반말 spectrum
- Particle-heavy for rhythm (도, 는데, 나)
- Native Korean 70%+ (Sino-Korean = corporate, not hip)
- Cultural refs: Korean memes, YouTube culture, streaming
- Structure: Conversational, almost speech-like

### **Critical Rule: Don't Copy English Hip into Korean**

**Don't:**
```
English: "We literally cannot stop innovating"
Korean (bad): "우리는 정말로 혁신을 멈출 수 없어요" 
❌ Looks translated. Not actually hip.
```

**Do:**
```
English: "We literally cannot stop innovating"
Korean (good): "새로운 거 없으면 우리가 못 살아요"
✓ Captures "addicted to innovation" in native, C8 voice
```

### C8 Localization Checklist

- [ ] Korean headline doesn't use English words arbitrarily ("프로" instead of "professional")
- [ ] No Sino-Korean overuse (check: >80% = corporate, not hip)
- [ ] 해체/반말 feels natural, not forced (test: would you say this IRL?)
- [ ] Particles create rhythm (read aloud)
- [ ] Cultural refs are Korean, not English transplant
- [ ] Meme/internet language used correctly (ask a young Korean to QC)
- [ ] Not trying to be cool; trying to sound authentic

## Final Checkpoint

Before shipping localized copy in both languages:

1. **Equivalence Test:** Do both versions communicate the brief? (Not word-for-word, but intent-level)
2. **Native Test:** Does a native speaker in each language say "this sounds natural"? Not "this is technically correct but sounds written"?
3. **Tone Test:** Does each version hit the tone template in its respective language?
4. **Culture Test:** Do references and idioms land in their respective culture?
5. **Rhythm Test:** Read aloud in each language. Does it flow?

If any fails → return to independent creation, don't translate.
