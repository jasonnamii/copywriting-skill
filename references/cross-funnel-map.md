# Cross-Funnel Map: Stage → Type → Theory → Tone

## 7-Stage Funnel Overview

```
[Awareness] → [Interest] → [Consideration] → [Conversion] → [Activation] → [Retention] → [Referral]
```

Each stage uses specific copy types and psychological frameworks. Tone shifts systematically as friction increases.

---

## Stage 1: AWARENESS

**Goal:** Reach audience, grab attention, build familiarity

### Active Copy Types
- **C5, C6, C8:** Ad Copy (Display), Social Post, Creator Hook
- **C1:** Email Subject (if cold outreach)
- **C3:** Landing Page Hero (if top-of-funnel paid traffic)

### Dominant Theory Framework
**AIDA (Attention-Interest-Desire-Action)** focused on A + I
- Attention: Hook in first 3 words
- Interest: Create curiosity gap (problem reframed)
- Psychology: Lizard brain activation (fear, pleasure, novelty)

### Recommended Tone
- **Funny:** +15 above type default (humor gets shares)
- **Casual:** At or +10 above default (approachable)
- **Irreverent:** At or +10 above default (challenge status quo)
- **Enthusiastic:** At type default (energy without desperation)

### Key Metrics
- Impressions, CTR (should be 1.5-3% for top-of-funnel)
- Shares/saves (social virality)
- Cost per click (should be lowest of funnel stages)

### C8 vs Universal Differences
| Aspect | C8 Mode | Universal |
|--------|---------|-----------|
| **Hook style** | "We built [shocking fact]" | "The problem with [category]..." |
| **Irreverence level** | Higher (challenge norms) | Moderate (respect conventions) |
| **Humor type** | Meme/cultural/absurdist | Clever/situational |
| **Energy** | High velocity, FOMO | Steady, reliable |

## Stage 2: INTEREST

**Goal:** Build credibility, deepen problem articulation, social proof

### Active Copy Types
- **C2, C3, C7:** Email Body, Landing Page (full page), Headline
- **B1, B3:** Positioning, Narrative frameworks
- **C4:** Search Ads (high intent signals)

### Dominant Theory Framework
**Uniqueness & Credibility (based on Cialdini + positioning)**
- Why us vs alternatives
- Proof: data, testimonials, shipped work
- Psychology: Reduce uncertainty (we are credible)

### Recommended Tone
- **Funny:** -5 from default (less humor, more specificity)
- **Casual:** At default (maintain approachability)
- **Irreverent:** At default or +5 (time to show POV)
- **Enthusiastic:** -5 to -10 (reserve energy for later)

### Key Metrics
- Landing page time-on-page (should increase 30-50% vs awareness)
- Read completion rate (scroll depth)
- Social proof engagement (comments on testimonials)
- Email open rate (if part of interest stage)

### C8 vs Universal Differences
| Aspect | C8 Mode | Universal |
|--------|---------|-----------|
| **Proof type** | Shipped work, creator case studies | Industry benchmarks, expert endorsements |
| **Narrative style** | "We iterated 12 times..." | "Research shows..." |
| **Authority** | Competence + transparency | Expertise + results |

---

## Stage 3: CONSIDERATION

**Goal:** Address objections, compare alternatives, build ZOPA (Zone of Possible Agreement)

### Active Copy Types
- **B2, B4:** Value Prop, Objection Handle
- **C2:** Email Body (sequences)
- **C3:** Landing Page (comparison section)
- **C4:** Search Ads (branded queries, competitor keywords)

### Dominant Theory Framework
**Loss Aversion & Social Proof (Kahneman + Cialdini)**
- What you lose if you don't act
- Who else is doing this (FOMO + consensus)
- Specific comparisons (better than, not just good)
- Psychology: Reduce perceived risk

### Recommended Tone
- **Funny:** -10 from default (humor becomes distraction)
- **Casual:** -5 from default (formality increases, trust grows)
- **Irreverent:** -10 from default (respect alternatives, don't mock)
- **Enthusiastic:** -15 from default (matter-of-fact focus)

### Key Metrics
- Objection pages visited (on landing page)
- Comparison content engagement
- Email sequence completion rate
- Time to decision (days from first touch)

### C8 vs Universal Differences
| Aspect | C8 Mode | Universal |
|--------|---------|-----------|
| **Objection handling** | "If you're skeptical, here's why it works" | "Common concerns: 1) ... 2) ..." |
| **Comparison style** | Feature + philosophy diff | Feature matrix + ROI calc |
| **Risk reduction** | Trial period, refund promise | Industry certifications, warranties |

---

## Stage 4: CONVERSION

**Goal:** Remove final friction, create urgency, close sale

### Active Copy Types
- **A4:** Upgrade Prompt (in-product)
- **C3:** Landing Page CTA section
- **C1:** Email (final conversion email in sequence)
- **D3:** Win-Back (if re-engaging dormant)
- **B5:** Social Proof (testimonial-heavy final push)

### Dominant Theory Framework
**Commitment & Consistency (Cialdini) + Scarcity**
- Create moment of commitment
- Time-limited or inventory-limited signal
- Final objection removal (FAQ, FAQs, guarantees)
- Psychology: Hard stop, decision now

### Recommended Tone
- **Funny:** -15 from default (focus, not distraction)
- **Casual:** At default or -5 (professional, not stuffy)
- **Irreverent:** -15 from default (respect the moment)
- **Enthusiastic:** +10 to +15 (urgency, forward momentum)

### Key Metrics
- Conversion rate (primary)
- Cost per conversion
- Drop-off rate by step (form field, payment, etc.)
- Time to complete checkout

### C8 vs Universal Differences
| Aspect | C8 Mode | Universal |
|--------|---------|-----------|
| **Urgency signal** | "Spots filling up" (true FOMO) | "Offer ends Friday" (deadline) |
| **CTA copy** | "Let's go", "Ship it" | "Get started", "Sign up" |
| **Objection** | "Worried? 30-day refund." | "100% satisfaction guaranteed" |
| **Final push tone** | Peer peer-to-peer urgency | Professional confidence |

## Stage 5: ACTIVATION

**Goal:** Get users to first success moment, reduce time-to-value

### Active Copy Types
- **A2:** Onboarding
- **A6:** Empty State
- **A8:** In-App Message
- **D1:** Welcome Series (email onboarding)
- **B2:** Value Prop (restated in-product context)

### Dominant Theory Framework
**Behavior Change & Habit Formation (BJ Fogg)**
- Prompt → Behavior → Reward (tight loop)
- Reduce effort, increase motivation
- Celebrate small wins (dopamine hits)
- Psychology: Build momentum

### Recommended Tone
- **Funny:** +10 from default (celebrate, build culture)
- **Casual:** +10 from default (friendly, peer tone)
- **Irreverent:** At default (normalize breaking rules)
- **Enthusiastic:** +10 from default (every step is a win)

### Key Metrics
- Onboarding completion rate (% who finish all steps)
- Time to first use (hours/days from signup)
- Feature adoption (% who try 3+ features)
- Aha moment hit rate (critical event completed)

### C8 vs Universal Differences
| Aspect | C8 Mode | Universal |
|--------|---------|-----------|
| **Step style** | "Invite 1 person, see magic" | "Step 1: Add your first project" |
| **Celebration copy** | "You shipped 🎉" | "Project created" |
| **Tone shift** | Friend showing cool trick | Trainer guiding exercise |
| **Pace** | Fast (3-5 min total) | Measured (allow exploration) |

## Stage 6: RETENTION

**Goal:** Create habit loops, prevent churn, increase frequency/depth

### Active Copy Types
- **A1, A3, A7, A8:** Feature Launch, Retention, Push, In-App
- **C1, C2:** Email (regular engagement), Email Body
- **D1, D2:** Welcome Series, Re-Engagement (after drop-off)
- **B3:** Narrative (storytelling to deepen connection)

### Dominant Theory Framework
**Habit Formation & Progress Loops (Fogg + Eyal)**
- Variable rewards (unpredictability keeps engagement)
- Progress tracking (show improvement, milestones)
- Community/belonging (social retention)
- Psychology: Make quitting harder than staying

### Recommended Tone
- **Funny:** At default or +5 (humor signals insider culture)
- **Casual:** At default (feel like community, not service)
- **Irreverent:** At default or +5 (we're different from others)
- **Enthusiastic:** +5 (celebrate usage, small wins)

### Key Metrics
- DAU/MAU (daily/monthly active users)
- Churn rate (% who stop using)
- Feature adoption depth (# features used)
- NPS (Net Promoter Score)
- Engagement loops completed (push → action)

### C8 vs Universal Differences
| Aspect | C8 Mode | Universal |
|--------|---------|-----------|
| **Trigger style** | "Your co-creators shipped 5 things" | "Your team has 3 pending tasks" |
| **Celebration** | "You're in the top 10% of shippers" | "Great work this week" |
| **Community signal** | Leaderboard, creator stats | Team progress dashboard |
| **Depth incentive** | More advanced features, new tools | More customization options |

## Stage 7: REFERRAL

**Goal:** Turn users into promoters, create viral loop

### Active Copy Types
- **C6, C8:** Social Post (share template), Creator Hook
- **A7:** Push (invitation prompt)
- **C1:** Email (referral program details)
- **B5:** Social Proof (user testimonials, case studies)

### Dominant Theory Framework
**Viral Loops & Reciprocity (Cialdini)**
- Easy share (one-click referral link)
- Incentive structure (clear reward for both sides)
- Social proof (show others who've referred)
- Psychology: Make it feel good to share

### Recommended Tone
- **Funny:** +15 from default (sharing should be fun)
- **Casual:** +15 from default (peer-to-peer, not corporate)
- **Irreverent:** +10 from default (empower users to break rules)
- **Enthusiastic:** +20 from default (celebrate growth, community)

### Key Metrics
- Referral rate (% who invite others)
- Viral coefficient (# new users per user who refers)
- Share completion (% who finalize referral link)
- Referral conversion rate (% of shares → signups)

### C8 vs Universal Differences
| Aspect | C8 Mode | Universal |
|--------|---------|-----------|
| **Share message** | "I'm shipping 3x faster now. Join." | "This tool has saved us hours." |
| **Incentive** | Status/community (creator tier upgrade) | Discount, extended trial |
| **Social signal** | "X creators using this" | "Trusted by Y companies" |
| **Tone** | Proud peer, casual flex | Professional recommendation |

## Cross-Stage Transition Rules

### Handoff Logic (How copy hands off between stages)

**Awareness → Interest**
- Drop novelty/surprise, deepen problem understanding
- Same brand voice, more specificity
- Shift from "hook" to "thesis"

**Interest → Consideration**
- Facts replace emotion
- Confidence replaces curiosity
- Offer alternatives (show you're honest)

**Consideration → Conversion**
- Remove optionality (clear next step)
- Add urgency (time/scarcity)
- Celebrate imminent commitment

**Conversion → Activation**
- Shift from "why buy" to "how use"
- Tone becomes warmer (peer → friend)
- Timeline shortens (minutes not hours)

**Activation → Retention**
- Shift from "first" to "frequent"
- Focus on habit, not novelty
- Introduce community signals

**Retention → Referral**
- Turn satisfaction into evangelism
- Make sharing easy, celebration assured
- Shift beneficiary from "you" to "your friends"
