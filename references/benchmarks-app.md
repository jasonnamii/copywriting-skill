# Benchmarks — App Internal Copy

> Mode tags: 🎨 C8 mode | 🌐 Universal | 🔄 Both

## A1 Onboarding Flow

### Slack — Welcome onboarding message
- **Copy:** "Slack brings your team's communication together. Set up your workspace and start collaborating."
- **Language:** EN
- **Mode:** 🌐
- **Why it works:** Frames onboarding as team enablement, not feature dumping; focuses on end benefit (collaboration) over interface education.

### Toss (토스) — Account setup completion
- **Copy:** "금융의 모든 것, 토스에서 쉽고 간편하게" (All financial services, easy and convenient on Toss)
- **Language:** KR
- **Mode:** 🌐
- **Why it works:** Single promise statement—removes decision paralysis by claiming completeness; Korean cultural value of simplicity (간편함) drives adoption.

### Linear — Project creation onboarding
- **Copy:** "Create your first issue or import from another tool to get started."
- **Language:** EN
- **Mode:** 🎨
- **Why it works:** Offers two paths (create fresh or migrate); low-friction first action; hipster-friendly tone = creative-tool credibility.

## A2 Empty States

### Notion — Blank database initial state
- **Copy:** "Your database is empty. Start by adding data or import from a spreadsheet."
- **Language:** EN
- **Mode:** 🌐
- **Why it works:** Two-part instruction (direct + optional path); transforms anxiety (blank canvas) into agency; permissive tone ("start by").

### 무신사 (Musinsa) — Empty wish list
- **Copy:** "관심 상품이 없어요. 스타일을 발견해보세요." (No liked products yet. Discover your style.)
- **Language:** KR
- **Mode:** 🎨
- **Why it works:** Empathetic reframe (not "no items" but "discover"); drives engagement through exploration metaphor; aspirational language (스타일) for fashion-focused audience.

### VSCO — Empty gallery on first login
- **Copy:** "Your canvas awaits. Upload a photo to start." (implied minimal interface text)
- **Language:** EN
- **Mode:** 🎨
- **Why it works:** Artistic framing ("canvas") elevates user action; present-tense urgency ("awaits"); targets creative identity.

## A3 Error Messages

### Duolingo — Network connectivity error
- **Copy:** "Looks like you're offline. Check your connection and try again."
- **Language:** EN
- **Mode:** 🌐
- **Why it works:** Non-blaming ("looks like"), diagnostic hint, 2-step recovery; owes success to user, not app blame culture.

### Stripe — Declined payment state
- **Copy:** "Your card was declined. Please try another payment method or contact your bank."
- **Language:** EN
- **Mode:** 🌐
- **Why it works:** Clear diagnosis + two recovery paths (alternative or escalation); removes user guilt ("contact your bank" = external factor).

### Figma — File sync error
- **Copy:** "Something went wrong syncing this file. We're working on it."
- **Language:** EN
- **Mode:** 🎨
- **Why it works:** Honest transparency without jargon; positions Figma as ally ("we're working"); creative apps can afford personality here.

## A4 CTA Labels

### Arc (브라우저) — Add bookmark button
- **Copy:** "Add Boost" / "Add Sidebar"
- **Language:** EN
- **Mode:** 🎨
- **Why it works:** Proprietary terminology (Boost, Sidebar) builds in-group language; implies power/agency; avoids generic "Add" or "Save".

### Slack — Message composer action
- **Copy:** "Send" (single word, always visible)
- **Language:** EN
- **Mode:** 🌐
- **Why it works:** Minimal, directional, instant clarity; reflexive verb assumes intent; no friction.

### Superhuman — Archive email
- **Copy:** "Archive" / "Delete" (with keyboard shortcut hint "e" / "d")
- **Language:** EN
- **Mode:** 🎨
- **Why it works:** Power-user targeting; keyboard-first mentality; two clear states (keep vs. remove); pro-app credibility.

## A6 Push Notifications

### Slack — Direct message alert
- **Copy:** "@[person] sent you a message"
- **Language:** EN
- **Mode:** 🌐
- **Why it works:** Specific mention (not "you have a message"); names the actor; implies urgency through person, not emoji.

### Duolingo — Streak reminder
- **Copy:** "Your 7-day streak is in danger! Tap to practice now."
- **Language:** EN
- **Mode:** 🎨
- **Why it works:** Gamification + scarcity (streak endangered); emotional hook (don't lose it); one-tap action.

### 토스 (Toss) — Dividend notification
- **Copy:** "배당금이 나왔어요! 토스에서 확인해보세요." (Your dividend arrived! Check it out on Toss.)
- **Language:** KR
- **Mode:** 🌐
- **Why it works:** Good-news trigger; casual honorific (어요) builds relationship; CTA is exploration ("확인") not click pressure.

- **Mode:** 🌐

- **Language:** EN
- **Mode:** 🎨
- **Why it works:** Minimal toast; contextual (appears where action happened); optional navigation (click = power user move).

## A9 Legal / Compliance Copy

### Stripe — Cookie consent banner
- **Copy:** "We use cookies to improve your experience. By using our site, you agree to our use of cookies." [Accept / Manage Preferences]
- **Language:** EN
- **Mode:** 🌐
- **Why it works:** Benefit-first framing ("improve your experience"); soft choice ("manage" vs. forced binary); B2B audience expects clarity.

### Superhuman — Privacy notice in onboarding
- **Copy:** "Your inbox is encrypted. We can't see your emails."
- **Language:** EN
- **Mode:** 🎨
- **Why it works:** Trustworthy, not legal-jargon; negative statement emphasizes protection ("can't see"); assures creator-audience of privacy.

## A10 Screen-Level Messaging (Status, Mode Indicators)

### Linear — Board empty state context
- **Copy:** "No issues match this filter. Try adjusting your search criteria."
- **Language:** EN
- **Mode:** 🎨
- **Why it works:** Diagnosis + recovery action ("adjust"); empowers user as solver; implies filter is feature (not limitation).

### Duolingo — Lesson streak indicator
- **Copy:** "🔥 7-day streak" [displayed prominently at top of screen]
- **Language:** EN
- **Mode:** 🎨
- **Why it works:** Emoji + number = game-like; public display drives habit; visible scarcity (if broken, goes to 0).

### Headspace — Session start
- **Copy:** "Today's meditation: 10 minutes of calm focus." [with visual progress indicator]
- **Language:** EN
- **Mode:** 🌐
- **Why it works:** Specificity (today's, 10 minutes); benefit statement ("calm focus"); visual timer = commitment signal.

---

**Collection notes:**
- C8 mode examples prioritize creative identity, minimal friction, proprietary language, power-user respect
- Universal examples prioritize clarity, trust, multi-step recovery, guilt-free error states
- Korean apps show consistent patterns: simplicity (간편), discovery (발견), transparency (결정)
